# Phone Book Application

Advanced Programming Course Project (Python) - Semester 3

## Features

✅ **Separation of logic from UI** (as per course PDF)
- `logic.py` - Logic and data layer
- `main_gui.py` - GUI layer

✅ **Main Features**
- Add new contact
- Edit existing contact
- Delete contact
- Smart search
- Sort by name
- Auto-save to CSV file
- Load from CSV file

✅ **Advanced Features**
- Phone number validation
- Prevent duplicate names
- Right-to-left Persian UI
- Status bar for messages

## How to Run

```bash
python main_gui.py
